# Tenagn Voice-First Design

Tenagn supports users who prefer speaking instead of typing.

Initial command families:

- Search merchants and products.
- Add item to cart.
- Read basket.
- Confirm checkout.
- Track order.
- Contact support.

Example:

```text
User: HabeshaGo, find me kitfo near Bole.
App: Three options found. Number one: Addis Chefs, 350 ETB. Say one to add it to cart.
```

Voice checkout must confirm the merchant, items, quantities, total, delivery address, and payment method before creating an order.
